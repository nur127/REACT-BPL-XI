# 🏏 Cricket Dream Team

A modern and responsive React application where users can build their own dream cricket team by selecting players within a virtual budget. The project features an attractive UI, player cards, dynamic team selection, and real-time coin balance management.

## 🌐 Live Demo

🔗 https://gruesome-tent.surge.sh/

---

## ✨ Features

- 🏏 Browse a list of cricket players with detailed information.
- 💰 Purchase players using a virtual coin balance.
- 👥 Add players to your selected team.
- ❌ Remove players from the selected team.
- ⚠️ Prevent duplicate player selection.
- 📊 Display player rating, role, nationality, batting style, bowling style, and price.
- 📱 Fully responsive design for desktop and mobile devices.
- 🎨 Clean UI built with DaisyUI and Tailwind CSS.

## 🚀 Installation

Clone the repository

```bash
git clone <your-repository-url>
```

Navigate to the project

```bash
cd <project-folder>
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
